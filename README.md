# نظام إدخال الأسماء — تطبيق ويب لملف Excel

تطبيق ويب بصفحة واحدة (Single File HTML) لإدخال بيانات الطلاب وتصديرها مباشرة إلى Excel. بُني ليكون أداة إنتاجية لمدخلي البيانات في المدارس والمؤسسات التعليمية.

---

## الميزات

### إدخال البيانات
- نموذج إدخال كامل بـ 13 حقلاً وفقاً لبنية Excel المستهدفة
- مسلسل تلقائي (غير قابل للتعديل)
- القوائم المنسدلة للجنس (ذكر/أنثى) والديانة (إسلام/مسيحية/أخرى)
- تحقق فوري (Validation) لكل حقل مع رسائل خطأ مضمنة
- التحقق من اسم الأم: يجب أن يحتوي على 3 أجزاء على الأقل

### الحقول المتكررة
عند إضافة صف جديد، الحقول التالية تبقى محفوظة (لا تُمسح):
- كود المحلية، كود المدرسة، اسم المدرسة، المحلية، اسم المدخل

مع زر "مسح" فردي لكل حقل عند المرور عليه بالماوس.

### الإقتراحات الذكية (Autocomplete)
أثناء الكتابة في الحقول المتكررة، تظهر قائمة منسدلة بأحدث 8 قيم مشابهة من الإدخالات السابقة — اختيارها يملأ الحقل تلقائياً.

### الجدول والسجل
- جدول يعرض جميع الصفوف المدخلة بترتيب الأعمدة نفسه الموجود في Excel
- كل صف يحتوي على زرّي **تعديل** و **حذف**
- التعديل: يملأ النموذج ببيانات الصف، يتغير زر الإضافة إلى "تحديث"
- الحذف: يطلب تأكيداً عبر نافذة منبثقة (مودال)

### البحث والتصفية
- شريط بحث فوق الجدول يصفّي النتائج فوراً في جميع الحقول
- اختصار لوحة المفاتيح: `/` للتركيز على البحث
- عند التصفية، زر التصدير يتغير إلى "تصدير المصفى (عدد)"

### التصدير إلى Excel
- زر "تصدير الكل (عدد)" — يصدّر جميع الصفوف
- زر "تصدير المصفى (عدد)" — يصدّر فقط الصفوف الظاهرة في نتائج البحث
- إسم الورقة: `ادخال ابتدائي 2025م` (مطابق للملف الأصلي)
- رؤوس الأعمدة بالعربية
- إتجاه الورقة: RTL أو LTR (زر تبديل بجانب زر التصدير)
- إسم الملف: `نظام_ادخال_الاسماء_مُعبَّأ.xlsx`

### الإستيراد من Excel
- زر "استيراد Excel" — يقرأ ملف .xlsx أو .xls
- يستخرج الصفوف من الأعمدة بالترتيب
- يتجاوز الرأس والصفوف الفارغة
- يضيف البيانات للجلسة مع حفظ للـUndo

### كشف التكرار
عند إضافة أو تعديل صف، إذا وُجد طالب بنفس الاسم الثلاثي (الأول + الثاني + الثالث):
- تظهر نافذة تأكيد (مودال) مع رسالة تحذير
- يمكن للمستخدم تأكيد الإضافة (تكرار) أو إلغائها

### التراجع (Undo)
- اختصار `Ctrl+Z` يمكن التراجع عن آخر 20 عملية (إضافة/حذف/مسح/استيراد)
- شريط تراجع يظهر بعد الحذف لمدة 6 ثوانٍ مع زر تراجع سريع

### الإحصائيات السريعة
شريط إحصائيات يظهر:
- عدد الصفوف الكلي
- عدد الذكور / الإناث
- اسم المحلية الأكثر تكراراً

### آخر الإدخالات
شريط بين النموذج والجدول يعرض آخر 5 أسماء طلاب تم إدخالها — بصرياً سريعاً لتأكيد الإدخال.

### تخزين البيانات
- حفظ تلقائي في `localStorage` بعد كل عملية
- تحذير عند الاقتراب من 4000 صف (الحد التقديري ~5000 صف)
- بقاء البيانات بعد تحديث الصفحة أو إغلاق المتصفح

### أعمدة ثابتة (Sticky Columns)
عند التمرير الأفقي للجدول، أوّل عمودين (مسلسل + كود المحلية) يثبتان مكانهما ليبقيا مرئيين دائماً.

### توجيه واجهة RTL
الواجهة كاملة باللغة العربية وبإتجاه من اليمين إلى اليسار.

## طريقة الاستخدام

### 1. فتح التطبيق
افتح الملف `نظام_ادخال_الاسماء.html` في المتصفح. لا حاجة لأي خادم (Server).

### 2. إدخال البيانات
1. إملأ الحقول في النموذج العلوي
2. جميع الحقول إلزامية عدا ما يُشار إليه
3. إضغط `Enter` أو زر **إضافة**
4. الحقول المتكررة تبقى — غيّر البيانات الخاصة بالطالب فقط وأضف التالي

### 3. تعديل أو حذف
- زر **تعديل** في الجدول: يملأ النموذج ببيانات الصف
- زر **حذف** في الجدول: يحذف الصف بعد التأكيد

### 4. البحث
1. إضغط `/` أو انقر على حقل البحث
2. إكتب أي كلمة — الجدول يصفّي فورياً

### 5. التصدير
1. إختر إتجاه الورقة (RTL/LTR) إذا لزم الأمر
2. إضغط **تصدير الكل** أو **تصدير المصفى**
3. الملف يُحمّل تلقائياً

### 6. الإستيراد
1. إضغط **استيراد Excel**
2. إختر ملف .xlsx أو .xls
3. الصفوف تُضاف إلى الجلسة الحالية

## إختصارات لوحة المفاتيح

| الإختصار | الوظيفة |
|----------|---------|
| `Enter` | إرسال النموذج (إضافة/تحديث) |
| `Ctrl+Z` | تراجع عن آخر عملية |
| `/` | التركيز على حقل البحث |
| `Esc` | إلغاء التعديل |
